

The Campus Recruitment Prediction System is a machine learning-based web application developed using the Flask framework in Python. This system predicts whether a student will be placed or not placed during campus recruitment and also estimates the expected salary based on the student's skills and academic profile.

The main goal of this project is to help students understand their placement chances and identify the domains or skills they need to improve in order to increase their chances of getting placed.

Machine Learning Models Used

The following machine learning models were trained and evaluated:

Decision Tree

Random Forest

Regression Model

Among these models, the Random Forest algorithm achieved the highest accuracy, so it was selected as the final model for prediction.

Features::

Predicts whether a student will be placed or not placed

Predicts the expected salary package

Suggests domains or skills to improve if the student is predicted as not placed

User-friendly web interface using Flask

Real-time predictions based on user input
